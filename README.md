## Printer Setup

- **Printer:** BambuLab P1S
- **Nozzles:** 0.2 mm · 0.4 mm · 0.6 mm
- **Primary slicer:** Bambu Studio
- **Secondary slicer:** OrcaSlicer
- **Filament diameter:** 1.75 mm
- **Filament source:** [india.numakers.com](https://india.numakers.com)

## Presets

Ready-to-import Bambu Studio `.json` profiles are in the [`presets/`](presets/) folder. Import via **Filament → gear icon → Import from file** in Bambu Studio.
